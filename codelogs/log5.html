Code Log 5: Images

  1. What are the differences between presenting an image using an HTML tag (like <img>) and presenting it as the background of an HTML element (using the CSS property background-image)? Think in terms of differences in code as well as differences in functionality and layout.

    Using the <img> tag puts the image inside the HTML content. It affects the layout, can use alt for accessibility, and is important for SEO. Using background-image is more for decoration, because it doesn’t affect the layout or structure, has no alt text, and is used to make the design look good.

  2. Besides background-image, what other CSS properties affect the display of background images? Describe at least two more CSS properties.
    • background-size: controls how big the image is, for example cover makes it fill the element and contain shows the full image without cutting it.
    • background-repeat: sets if the image repeats or not (for example no-repeat stops it from repeating).
