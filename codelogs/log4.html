Code Log 4: CSS Color

What do these CSS properties do?
  • color: Defines the color of the text inside an element. Example: p { color: blue; }
  • background-color: Sets the background color of an element. Example: div { background-color: lightgray; }
  • border-color: Specifies the color of an elementâ€™s border. Example: div { border: 2px solid red; border-color: red; }
How does CSS accept color as a value? Provide examples of colors in these styles:
  • Named: Use predefined color names. Example: p { color: blue; }
  • Hexadecimal: Use a # followed by numbers and letters that represent red, green, and blue values. Example: p { color: #ff0000; } (red)
  • RGB: Define colors using RGB values (red, green, blue). Example: p { color: rgb(0, 255, 0); } (green)
  • RGBA: Uses red, green, blue, and alpha (transparency) values. Example: p { color: rgba(255, 0, 0, 1); } (solid red)
